Red WiFi No Segura Detectada

Esta red inalámbrica no cuenta con mecanismos de seguridad adecuados. Tu información personal podría estar en riesgo.

  • Robo de contraseñas
  • Interceptación de datos
  • Acceso no autorizado a información privada

Este escenario es una simulación académica utilizada para demostrar los riesgos de conectarse a redes WiFi abiertas.

¿Por qué esta red es peligrosa?

Las redes WiFi abiertas no utilizan cifrado, lo que permite que un atacante pueda interceptar la información transmitida por los dispositivos conectados.

Ataques comunes incluyen suplantación de redes, captura de tráfico y manipulación de datos.

Este sistema no recopila ni almacena datos personales.
Proyecto educativo con fines académicos.
